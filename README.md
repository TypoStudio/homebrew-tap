# TypoStudio Tap

## How do I install these formulae?

`brew install typostudio/tap/<formula>`

Or `brew tap typostudio/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "typostudio/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
